# BCS Lv 4 BootCamp - Project 7 - FullStack Notes Taking Application Using "Nodes" and "Express"

## Description
This is a simple Note Taking and Tracking Application, with responsive web application for managing notes effectively. User can add, edit, delete, complete, and restore notes, with separate viewing page for active, deleted and completed notes. The app supports dark mode compatibility, and provides a clean, user-friendly interface with modals for confirmation of editing, permanently deleteing and completed notes (with ability to reactivate notes if deemed not completed in full for modification).

---

## Technologies Used
1. HTML 5; 
2. CSS 3; 
3. JavaScript (ES6+); 
4. Bootstrap 5; 
5. Font Awesome; 
6. Local Storage - Used for darkmode persistence across all page (remember user setting preferences); 
7. Fetch API - Used for backend integration.

---

## Usage
1. **Active Notes:** Add new notes, editing existing notes, marking for completion, or delete notes (soft delete - move notes to Deleted Notes Page); 
2. **Deleted Notes:** Ability to restore notes back to active or permanently deleting them (removed from server); 
3. **Completed Notes:** Allow for reactivation of completed notes back into active listing - for modification and/or if notes parameters no longer deemed completed; 
4. **Dark Mode:** - Allow for accessibility setting and toggle thememing based on user preference using sidebar switch - with persistence across all pages to maintain user setting when reloads.

---

## How To Use
1. Before usage of this application/software/repo, please contact Tri Ng, by following the steps prescribed in Licenses.md; 
2. Clone or download the repo into your local working folder; 
3. Open "index.htm" in your browser; 
4. Use the sidebar to navigate between Current (Active), Deleted, or Completed Notes; 
5. Add a note by entering a Title, and optional description, then click **Add Note**; 
6. Edit a note by clicking the **Edit** button; preview window will appears for any changes before confirmation is required; 
7. Delete a note to move it to Deleted Notes; confirmed pernament deletion here if desired; 
8. Completed a note to move it to Completed Notes; reactivation of completed notes using the **Mark Active** button; 
9. To toggle Dark Mode, use the appropriately iconed button in the side bar as needed/preferred.
